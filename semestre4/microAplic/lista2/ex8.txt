Esta linha de instrucao ainda não funcionará como esperado. Para conseguir escrever o valor do acumulador no endereço zero da memoria ainda é necessário conectar devidamente o a memória no 8051. Para isso, devemos adicionar um circuito com 8 Latchs, 74LS373, para armazenar os valores menos significativos do endereço (Apesar de ter o mesmo resultado se ligar todos os pinos de endereço da memória no GND). Após isso podemos conectar os 8 pinos menos significativo do endereço da saida do 74LS373 na memória RAM (se quiser utilizar mais memoria ainda podemos utilizar alguns fios para conectar o A8-A14 do P2).
Finalmente, ainda é necessário conectar o WR do 8051 no WR da memoria. Se quisermos operacao de read podemos ligar o RD também, ou deixar em 1 caso contrário. Se os pinos de dados da memória estiverem conectados no P0 já deve funcionar como esperado.
